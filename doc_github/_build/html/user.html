User Guide¶

This guide is a resource for installing the dipde package. It is maintained by the Allen Institute for Brain Science.

Quick Start Install Using Pip¶

  1. Pip install for a single user:

    pip install git+http://stash.corp.alleninstitute.org/scm/~nicholasc/dipde2.git --user

Install Using Setup Tools¶

  1. Download the distribution.

    TODO

  2. Unpack the distribution.

    tar xvzf dipde-0.1.1.tar.gz

  3. Install using setuptools

    cd dipde-0.1.1 python setup.py install --user

Uninstall¶

  1. Simply use pip.

    pip uninstall dipde

Required Dependencies¶

  • NumPy
  • SciPy
  • MatPlotLib
  • SymPy
